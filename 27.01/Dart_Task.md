# Dart: 100 заданий по особенностям языка

## Раздел 1: Переменные и типы данных (Задания 1-20)

### Базовые переменные и типизация

**1.** Создайте переменную типа `String` и выведите её значение в консоль. Значение: "Привет, Dart!"

**2.** Объявите переменную типа `int` с числовым значением 42 и выведите её.

**3.** Создайте переменную типа `double` со значением 3.14 и выведите её.

**4.** Объявите переменную типа `bool` со значением `true` и выведите результат.

**5.** Создайте переменную типа `List<int>` и добавьте в неё числа от 1 до 5.

**6.** Объявите переменную типа `Map<String, int>` для хранения имён и возрастов людей.

**7.** Создайте переменную типа `Set<String>` с уникальными значениями: "яблоко", "апельсин", "груша".

**8.** Объявите переменную с использованием ключевого слова `var` и присвойте ей значение строки.

**9.** Используя `var`, создайте переменную для числового значения 100.

**10.** Напишите код, который покажет, какой тип был вывернут для переменной `var` при инициализации строкой.

### Ключевое слово `dynamic`

**11.** Создайте переменную типа `dynamic` и поочередно присвойте ей целое число, строку и логическое значение.

**12.** Объявите переменную `dynamic` и попробуйте выполнить с ней недопустимую операцию (например, деление строки на число). Объясните, почему это возможно.

**13.** Создайте переменную `dynamic` и проверьте её тип во время выполнения программы с помощью `runtimeType`.

**14.** Напишите функцию, которая принимает параметр типа `dynamic` и проверяет его тип внутри функции.

**15.** Сравните поведение `var` и `dynamic` на примере: измените тип переменной после объявления для обоих ключевых слов.

### Ключевое слово `final`

**16.** Создайте переменную `final` с начальным значением и попробуйте изменить её (объясните результат).

**17.** Объявите `final` переменную типа `List<String>` и добавьте элементы в список.

**18.** Создайте `final` список и попробуйте переприсвоить переменную новым списком. Объясните поведение.

**19.** Объявите `final` переменную без инициализации и инициализируйте её позже в коде.

**20.** Напишите класс с `final` свойствами и попытайтесь изменить их после создания объекта.

### Ключевое слово `const`

**21.** Создайте `const` переменную и выведите её значение в консоль.

**22.** Объявите `const` переменную и попробуйте изменить её (объясните ошибку компиляции).

**23.** Создайте `const` список и попробуйте добавить элемент в него. Объясните результат.

**24.** Напишите код с `const` переменной, которая инициализируется значением, известным только во время выполнения.

**25.** Сравните `final` и `const` переменные на практике. Что разрешено для каждого типа?

### Ключевое слово `late`

**26.** Объявите переменную `late String` и инициализируйте её позже в функции `main()`.

**27.** Создайте переменную `late int` и выведите её значение после инициализации.

**28.** Попробуйте использовать переменную `late` до её инициализации. Объясните ошибку.

**29.** Объявите переменную `late final` и объясните её поведение.

**30.** Напишите код, где `late` переменная инициализируется внутри условного блока `if`.

### Комбинированные параметры

**50.** Напишите функцию, где параметр может быть позиционным или именованным.
